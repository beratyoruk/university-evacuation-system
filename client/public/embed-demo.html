Tahliye Asistanı Widget Demosu Demo

Bu sayfa, üniversite sitelerine tek satırda eklenebilen tahliye widget'ını gösterir. Sağ alt köşedeki yeşil butona tıklayın.

Nasıl Çalışır?

Widget, herhangi bir HTML sayfasına tek satır script etiketi ile eklenir. Sayfa yüklendiğinde sağ alt köşeye yeşil 🚪 Tahliye butonu yerleştirir. Ziyaretçi butona tıkladığında, tahliye asistanı tam ekran iframe olarak açılır.

Entegrasyon Kodu

</body> etiketinden hemen önce şu satırı ekleyin:

<script src="https://your.domain.com/widget.js"
        data-university="itu"
        data-building="main"
        data-color="#16a34a"
        data-position="bottom-right"></script>

Parametreler

  • data-university — Üniversite slug'ı (gerekli)
  • data-building — Varsayılan bina (opsiyonel)
  • data-host — Sunucu URL'i (opsiyonel; script URL'inden çıkarılır)
  • data-color — Buton arka plan rengi
  • data-text-color — Buton yazı rengi
  • data-label — Buton metni (varsayılan: 🚪 Tahliye)
  • data-position — bottom-right / bottom-left / top-right / top-left

JavaScript API

Widget yüklendikten sonra window.UEWidget üzerinden programatik olarak kontrol edilebilir:

UEWidget.open();   // modalı aç
UEWidget.close();  // modalı kapat
UEWidget.config(); // yapılandırmayı döndür

Olay Dinleme

Sayfa, ue:emergency CustomEvent'ini dinleyerek acil durum bildirimlerini yakalayabilir:

window.addEventListener("ue:emergency", (e) => {
  console.log("Acil durum:", e.detail);
});
